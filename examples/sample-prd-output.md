# Sample PRD Output

## Problem
Planning inputs are fragmented, which slows down product definition and creates ambiguity for execution teams.

## Goal
Create an AI-assisted workflow that turns scattered planning context into structured PRDs and delivery artifacts.

## Success Metrics
- reduce PRD draft time
- improve handoff clarity
- reduce dependency surprises
- improve exec update quality

## Risks
- low-quality source inputs
- over-automation without human review
- unclear ownership of downstream actions
